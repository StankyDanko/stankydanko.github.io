# StankyDanko — Personal Portfolio

## Current State
Live at stankydanko.github.io. React 19 + TypeScript + Tailwind 3.4 + Framer Motion. Synthwave dark aesthetic, JetBrains Mono font. Single-page with headliners, garden grid, album player. Auto-deploys via GitHub Actions.

## What's Done
- 3 headliner cards (Naptime's Over, OMNI, ZenoType)
- Garden grid (SouthernSky, Star Catcher, video-fx, Local AI Rig)
- In-app album player (17-track Naptime's Over)
- Canvas animations (MyceliumNetwork, FooterCanvas, SynthwaveLines)
- Framer Motion scroll reveals + page transitions
- Brand color system (dark/green/purple/cyan/orange/blue)
